# AEGIS Flutter App

> Offline-first P2P mesh communication — no internet, no server required.

See the main [README](../README.md) for full documentation including:
- How devices connect (all 4 methods)
- Architecture and mesh routing
- Demo script
- Troubleshooting

## Quick Start

```bash
flutter pub get
flutter run
```

Requires Android device (API 21+). Grant Location + Bluetooth + Nearby permissions on first launch.
